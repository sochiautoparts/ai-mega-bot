## 🚀 Быстрый старт (5 минут)

### Шаг 1: Создай бота в Telegram

```
1. Открой @BotFather в Telegram
2. Отправь /newbot
3. Придумай имя: "AI Ассистент"
4. Придумай username: "your_ai_assistant_bot"
5. Скопируй токен вида: 1234567890:ABCdef...
```

### Шаг 2: Получи OpenAI API ключ

```
1. Зайди на https://platform.openai.com/api-keys
2. Нажми "Create new secret key"
3. Скопируй ключ (начинается с sk-)
4. Стоимость: ~$0.15 за 1M токенов (gpt-4o-mini)
   → Примерно $1 на 500-1000 запросов пользователя
```

### Шаг 3: Узнай свой Telegram ID

```
1. Открой @userinfobot в Telegram
2. Отправь любое сообщение
3. Скопируй свой ID (число вида 123456789)
```

### Шаг 4: Настрой проект

```bash
# Клонируй или скопируй проект
cd telegram-ai-bot

# Создай .env файл
cp .env.example .env

# Отредактируй .env — вставь свои ключи:
# BOT_TOKEN=1234567890:ABCdef...        (от BotFather)
# OPENAI_API_KEY=sk-...                  (от OpenAI)
# ADMIN_IDS=123456789                    (твой Telegram ID)
```

### Шаг 5: Запусти бота

**Вариант А — Локально (для тестирования):**
```bash
pip install -r requirements.txt
python bot.py
```

**Вариант Б — Docker (для продакшена):**
```bash
cd deploy
docker-compose up -d
```

**Вариант В — Бесплатный хостинг:**
- **Render.com** — бесплатный tier, автодеплой с GitHub
- **Railway.app** — $5 бесплатных кредитов/мес
- **PythonAnywhere** — бесплатный tier для Python

## 💳 Настройка Telegram Stars (оплата)

Telegram Stars — встроенная платёжная система Telegram. Пользователи платят звёздами, ты получаешь реальные деньги.

### Активация:
```
1. Открой @BotFather
2. Отправь /mybots
3. Выбери своего бота
4. Payments → Telegram Stars → Connect
5. Готово! Оплаты автоматически обрабатываются
```

### Вывод средств:
- Звёзды можно конвертировать в TON (криптовалюта)
- Минимум для вывода: 1000 Stars
- Комиссия вывода: зависит от рынка

## 🔧 Кастомизация

### Изменить цены:
Отредактируй `.env`:
```
PRICE_STARS_MONTH=350    # поднять до $3.50
PRICE_STARS_3MONTH=800   # поднять до $8
PRICE_STARS_YEAR=2500    # поднять до $25
```

### Изменить лимиты:
```
FREE_CHAT_LIMIT=3        # уменьшить — больше купят
FREE_IMAGE_LIMIT=1       # минимум — сильнее мотивация
FREE_COPYWRITE_LIMIT=2
```

### Изменить модель AI:
```
OPENAI_MODEL=gpt-4o           # мощнее, но дороже
OPENAI_MODEL=gpt-4o-mini      # дешёвая и быстрая (рекомендую)
OPENAI_IMAGE_MODEL=dall-e-3   # лучшее качество
OPENAI_IMAGE_MODEL=dall-e-2   # дешевле ($0.016/картинка)
```

### Добавить YooKassa (для рублёвых платежей):
```python
# В handlers/payment.py замените provider_token:
provider_token="ваш_токен_yookassa"
currency="RUB"
prices=[LabeledPrice(label="Pro 1 мес", amount=25000)]  # 250₽ в копейках
```

## 🆘 Частые проблемы

**Бот не отвечает:**
- Проверь BOT_TOKEN в .env
- Убедись что бот запущен без ошибок в логах

**AI не отвечает:**
- Проверь OPENAI_API_KEY
- Проверь баланс на platform.openai.com

**Оплата не работает:**
- Подключи Telegram Stars через @BotFather
- provider_token должен быть пустой строкой для Stars

**База данных не создаётся:**
- Проверь права на запись в директорию
- Удали старый .db файл и перезапусти
